# springmvc
SpringMVC学习记录

### 添加一键部署脚本
clone项目
```
# cd /home/wcy/
# mkdir projects
# cd projects
# git clone https://github.com/wangchenyan/learnspring.git
```
复制打包脚本
```
# cp /home/wcy/projects/learnspring/deploy /usr/local/bin/
```
添加可执行权限
```
# chmod +x /usr/local/bin/deploy
```
一键部署
```
# deploy
```
